Software:

The BC-780XLT has not yet been released however I assume it will have much the same command set as it predecessors the BC-245XLT and the BC-895XLT.

These products are known to work with the BC-245 and BC-895:

Senss BC-245 Computer Interface (Windows)
Uniden's SmartScanner (windows) Uses the modem on your pc to download frequencies.
BC245XLT Scanner Backup Utility (windows)
Bandscope (windows) - This software is actually meant for the BC-895 but the command sets are similiar.
Scancat Gold (windows)
Radio Manager for Windows (windows)
WinScan (windows)
Scanner Programming System (Windows) Works with WinScan - Preprograms scanner to known lists for your region of the country.
RadioMax supports the BC-895 so it probably would work with the 245 as well. (windows)
ScanMaster is another BC-895 control software thay may work with the BC-245.
AirScan (freeware)
e2a (DOS) - Converts ETrunk GRP files into Uniden AFS format
