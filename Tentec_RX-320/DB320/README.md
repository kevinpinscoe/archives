# DB-320 a database in Microsoft Windows for the Tentec RX-320 and RX-321

## Install

Only supported in Microsoft Windows originally in Win98 and Win 2000 but ay work in
Windows 11 I have not tried.

-Unzip the zip files into .cab (Microsoft Cabinet files)
-Extract the cab files (cabextract file.cab) in Linux or Use 7Zip in Windows
-Run setup.exe
